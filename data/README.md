# Dataset

## Source
Synthetic audiometric dataset generated by `data/generate_data.py`.

To use a real Kaggle dataset instead, download **"Hearing Test Results"** from
https://www.kaggle.com/datasets/paultimothymooney/hearing-test-results-dataset
and place the CSV inside `data/raw/audiometric_data.csv`.

## Features

| Column | Type | Description |
|---|---|---|
| age | int | Patient age (18–80) |
| gender | int | 0 = Female, 1 = Male |
| noise_exposure | float | Years of significant noise exposure |
| tinnitus | int | Tinnitus present: 0 = No, 1 = Yes |
| hearing_Xhz_left/right | float | Hearing threshold (dB HL) at freq X for each ear |
| hearing_loss_severity | int | **Target label** (see below) |

## Target Labels (WHO Classification)
| Label | Class | PTA Range |
|---|---|---|
| 0 | Normal   | ≤ 25 dB HL |
| 1 | Mild     | 26–40 dB HL |
| 2 | Moderate | 41–60 dB HL |
| 3 | Severe   | > 60 dB HL |

PTA = Pure Tone Average across 500, 1000, 2000, 4000 Hz (better ear).
